# Safety Notes

This platform should use careful language for sensitive media interpretation.

Allowed:
- object label
- scene label
- facial expression signal
- transcript sentiment
- repeated phrase
- contradiction indicator
- human review recommended
- confidence score

Avoid:
- liar
- high
- guilty
- drug user
- truthful
- impaired as a definitive conclusion
- criminal conclusion

Recommended display:

> This result is an automated signal only. It is not a medical, legal, psychological, or truthfulness determination. Human review is required for sensitive decisions.
